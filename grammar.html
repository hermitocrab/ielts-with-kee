P1/3: 答题神逻辑
😫
痛点：一紧张就只会说 Yes/No，脑子一片空白，或者像背书。
请充当我的雅思口语教练。现在的题目是：[在此处粘贴题目]。 我的简单想法/关键词是：[在此处输入你的真实想法，中文或简单的英文均可]。 请执行以下操作： 1. 使用“逻辑链条”方法（如：视情况而定法、细节爆炸法、推己及人法）将我的简单想法扩展成一个完整的雅思口语回答。 2. 极其重要：在回答中必须自然地嵌入至少2-3个Discourse Markers（话语标记词），例如表示思考的"Let me see..."，表示强调的"The thing is..."，或表示转折的"Having said that..."。 3. 请提供三个版本的回答： - 版本A（B1水平）：简单清晰，语法准确。 - 版本B（B2/C1水平）：加入地道习语和更复杂的句式。 - 版本C（Native Flow）：完全口语化，包含自然的犹豫和自我修正（如 "I mean...", "What I'm trying to say is..."）。 4. 最后，列出用到的所有“话语标记词”及其功能。

P2: 故事弧光法
请充当我的雅思口语Script Writer。 Task 2 题目卡是：[在此处粘贴Part 2 Topic]。 我想讲的真实经历是：[在此处用中文描述你的故事，越具体越好，例如：我去了那个公园，但是下雨了，不过雨中的景色很美]。 请帮我将这段经历改写成一篇2分钟的口语稿，要求： 1. 结构上使用“The Pop-in Strategy”（开场白策略）：以 "As soon as I saw this topic, the first thing that popped into my mind was..." 开头。 2. 逻辑上使用“Story Arc”（故事弧光法）：包含背景(Context) -> 冲突/转折(Conflict/Twist) -> 感受/结局(Resolution)。 3. 语言风格：请使用“Liquid Glass”风格，即词汇高级但听起来非常顺滑自然，避免生僻的书面词汇。 4. 在文中用【括号】标出哪里应该停顿、哪里应该重音、哪里应该加速。

G1: Nano 语法透视
🔬
Nano Banana 独家功能：长难句读不懂？让 AI 把它变成一张清晰的解剖图。
请充当我的语法分析师（Grammar Analyst）。 ⚠️ 警示：请务必勾选 “Tools - 🍌 Create images” 选项，否则无法生成分析图。 这句话难住了我：[粘贴长难句]。 请执行以下操作（必须包含中英双语输出 Bilingual Output）： 1. 🍌 Nano Visuals (语法树图解)： 请生成一张【SVG 矢量图】或【Mermaid 流程图】。 - 将句子拆解，在每个单词上方明确标注成分：S(主语), V(谓语), O(宾语)。 - 用颜色区分主句（骨架）和从句（皮肉）。 - 用箭头 ---> 展示修饰指向（例如：which引导的从句指向它修饰的名词）。 2. 📖 B1 词汇扫盲 (Vocabulary Check)： - 请找出句子中 B1 级别学生可能不熟悉的词汇或表达。 - 为这些词汇提供【中英双语】解释，并注释其在地道语境中的具体用法（Usage Note）。 3. 🔪 结构层级拆解： - 第一层（核心）：S + V + O (剥离所有修饰)。 - 第二层（血肉）：还原定语从句、状语从句，并中英双语解释它们的功能。 4. 🗣️ 自然翻译： - 指出这句话对中国学生来说最“反直觉”的地方。 - 给出地道的中文翻译和 B2 级别的英文改写。

Strict Examiner Mode
请充当严厉的雅思前考官。 题目是：[粘贴题目]。 这是我的回答录音转文字稿：[粘贴你的回答文本]。 请基于雅思口语评分标准（Band Descriptors）进行评分（0-9分）并给出严厉反馈： 1. Fluency & Coherence：我是否使用了足够的连接词？哪里出现了不自然的停顿？ 2. Lexical Resource：有没有中式搭配（Chinglish）？请提供3个更地道的同义替换（Idiomatic Expressions）。 3. Grammar：指出我所有的语法错误，并展示如何用更复杂的句型（如虚拟语气、倒装句）重写这一句。 4. 改写建议：请给我一个满分版本的示范，并解释为什么它更好。

W2: 逻辑体检中心
请充当雅思写作导师。 题目是：[粘贴大作文题目]。 我的核心观点是：[输入你的观点，例如：同意，因为这样可以保护环境]。 请帮我进行“逻辑体检”： 1. 批判性思维（Critical Thinking）：我的观点是否有逻辑漏洞？请列出反方可能攻击的点。 2. 论证拓展（Extension）：请使用“PEER模型”（Point观点 -> Explanation解释 -> Example举例 -> Result结果）帮我把这个观点发展成一个严密的段落。 3. 连接词检查：在段落中必须包含高阶的逻辑连接词（如 "Conversely", "Given the fact that...", "Examples include..."）。

W1: 学术词汇升级
请充当雅思写作Task 1专家。 这是一句我在描述图表时写的句子：[粘贴你的句子，例如：Number went up quickly]。 请帮我进行“学术化升级”： 1. 词汇替换：把简单的动词和副词替换为更精准的词汇（如 "surged", "plummeted", "marginally"）。 2. 句式多样性：请用三种不同的句型重写这句话（例如：主语做名词、时间状语前置、使用分词结构）。 3. 语境注明：解释这些替换词在描述数据时的细微差别（Nuance）。
🧪
Deep Learning: 词汇内化
我想学习这个词/短语：[输入单词]。 请不要只给我定义，请执行以下操作： 1. 定义：用简单的B1英语解释它。 2. 语境：它通常用于口语还是写作？是褒义还是贬义？ 3. 搭配（Collocations）：给出3个它最常用的固定搭配（Adjective + Noun 或 Verb + Noun）。 4. 替换：给出1个比它低级的词（A2）和1个比它高级的词（C1），并解释区别。 5. 造句：结合[此处输入你感兴趣的领域，如：科技/旅游/教育]，造一个地道的雅思例句。 6. 记忆钩子：给我一个助记技巧或者词源故事帮助记忆。
